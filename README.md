# EVERCURES – Nail Salon Landing Page Website

## Project Overview
I designed a modern and elegant Nail Salon Landing Page called **EVERCURES** that helps beauty businesses showcase their services, attract potential customers, and encourage online appointment bookings through a visually appealing and user-friendly website interface.

## Design Goal
The goal of this project was to create a premium landing page experience that highlights salon services, builds customer trust, and improves engagement through a clean, modern, and responsive design.

## Sections Designed
- Hero Section
- About Us Section
- Services Section
- Featured Treatments
- Customer Testimonials
- Pricing Section
- Contact & Booking Section
- Footer Section
- Interactive Prototype Flow

## Key UX Decisions
- Clear call-to-action buttons for bookings
- Simple and intuitive navigation
- Modern and elegant visual design
- Organized content structure
- Customer-focused user journey
- Responsive layout approach

## Color & Typography
- Elegant and beauty-inspired color palette
- Modern and readable typography
- Consistent spacing and visual hierarchy
- Professional branding elements

## Tools Used
- Figma
- Auto Layout
- Components
- Design System
- Icons & UI Assets

## Outcome
The final design delivers a professional online presence for a nail salon, making it easier for customers to explore services, view pricing, and book appointments while enhancing the brand's visual identity.
